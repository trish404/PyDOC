User Guide¶

To use the Text Processing Tool, provide the path to the text file and use the following options:

Examples:¶

  1. Count the words in a file: `bash python text_proc.py -f sample.txt -wc `

  1. Count the characters in a file: `bash python text_proc.py -f sample.txt -cc `

  1. Find a specific word in a file: `bash python text_proc.py -f sample.txt -find "example" `

  1. Replace a word in a file and save the output: `bash python text_proc.py -f sample.txt -r "old" "new" `

See Command-Line Tools for more details on all options.
